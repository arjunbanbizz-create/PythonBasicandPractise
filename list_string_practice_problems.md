# String Practice Problems

## Easy
1. Reverse a string
2. Check if a string is a palindrome
3. Count vowels in a string

## Medium
1. Find the first non-repeating character in a string
2. Remove duplicates from a string
3. Check if two strings are anagrams

## Hard
1. Implement a function to perform string compression
2. Find the longest substring without repeating characters
3. Given two strings, write a method to decide if one is a permutation of the other.
